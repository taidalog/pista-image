## About

PowerShell modules for simple image processing


## Requirements

- Windows PowerShell x.x
- PowerShell x.x
- .Net Framework x.x


## Import

1. Save entire pista-image directory in 'module path'.
1. On PowerShell console, type command below:
```
Import-Module pista-image -Force
```
